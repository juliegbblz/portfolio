# react-portfolio

This is a Vite + React conversion of the original static portfolio.

To install dependencies and run the dev server:

```bash
cd react-portfolio
npm install
npm run dev
```

If you need to copy assets (images, PDF) from the project root into this project public folder, run:

```bash
# from project root (/home/ju/Portfolio)
./react-portfolio/sync-assets.sh
```

Open the dev server URL shown by `npm run dev` (usually http://localhost:5173).
